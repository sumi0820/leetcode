## 気づいたこと/ memo

- どうやら head をそのまま使えるということが分かった
  - https://www.youtube.com/watch?v=kOhQ5bfpq2I を観る
    - fast は２つ先、slow は１つ先の値を参照するようにする
    - fast が null(=参照先がない)場合は false、slow === fast になったら true を返し, それまで iterate させる
  - Floyd’s Cycle Detection というらしい
  - 動画内で紹介されている記述法を TypeScript で書き直してみる

```typescript
function hasCycle(head: ListNode | null): boolean {
  // headがnullの場合があるのでearly return
  if (head === null) return false;

  let slow: ListNode | null = head;
  let fast: ListNode | null = head;

  while (fast !== null) {
    // fast.nextがnullの状態でfast.next.nextを試みると参照エラーになるためガード節を入れる
    if (fast.next !== null) {
      // fastを２つ先に進める
      fast = fast.next.next;
    } else {
      return false;
    }
    // slowを１つ進める（型エラーを避けるためにternary operatorを使用）
    slow = slow ? slow.next : null;

    // a cycle in the linked listの場合はtrue
    if (slow === fast) {
      return true;
    }
  }

  return false;
}
```

気づいたこと

- while 内の if 文が若干複雑で読みにくかったが、fast が２つ先に進めることも while 文の条件に加えることでシンプルになっている
- GPT の解説によると、2 ポインタ：slow=1 歩、fast=2 歩を使って slow === fast を比較する方法を Floyd 法（Floyd’s Cycle-Finding Algorithm）と言うらしい

解法

```typescript
function hasCycle(head: ListNode | null): boolean {
  if (!head || !head.next) return false;

  let slow: ListNode | null = head;
  let fast: ListNode | null = head;

  // fast が 2歩進める間だけ回す（null参照を防ぐ）
  while (fast && fast.next) {
    slow = slow!.next; // 1歩
    fast = fast.next.next; // 2歩
    if (slow === fast) return true; // 追いついた＝サイクルあり
  }
  return false; // どこかで終端に到達＝サイクルなし
}
```

気づいたこと

- Floyd とは別に Brent 法（Brent’s Cycle Detection Algorithm）というものがあり、それについてもコールドリーディングしてみた
  - Brent 法は「距離を指数的に伸ばしながら」比較するのが特徴

解法

```typescript
function hasCycle(head: ListNode | null): boolean {
  if (!head || !head.next) return false;

  let slow: ListNode | null = head;
  let fast: ListNode | null = head.next; // slowより１ステップ進めた状態が初期値
  let power = 1; // 比較間隔（＝次に slow を更新するまでの許容量）
  let lam = 1; // 現在の距離（＝サイクルを1周するのに必要なステップ。lambdaの略）

  while (fast) {
    if (slow === fast) return true;

    if (lam === power) {
      slow = fast; // slowを更新
      power *= 2; // 距離を2倍に拡大（以後、チェック間隔が指数的に伸びる（1,2,4,8,…））
      lam = 0; // 距離カウンタをリセット
    }

    fast = fast.next; // 1歩ずつ進む(Brent は 常に fast は1歩、比較間隔を広げて比較回数を減らす。)
    lam++; // slow からの距離を 1 増やす。fast が進んだ分を記録
  }

  return false; // nullに到達 → サイクルなし
}
```
